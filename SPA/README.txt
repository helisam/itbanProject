Pré-requisitos

- Ter o visual studio 2017 versão >= v15.7

- Ter restaurado a base de dados tables.sql em seu MsSql

- Instalar .NET Core 2.1 Preview 2 SDK
https://www.microsoft.com/net/download/thank-you/dotnet-sdk-2.1.300-preview2-windows-x64-installer

- Instalar ASP.NET Core Blazor Language Services extension
https://marketplace.visualstudio.com/items?itemName=aspnet.blazor
